HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the framework/building blocks of the webpage.
  • HTML is also used in allowing the user to easily navigate the page.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • There are three ways to implement CSS: Inline, internal, and external.
  • CSS is used to add style to your projects.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add [file]: adds a file to our next commit (Use -A to add all files!)
  • git commit -m "(description)": commits work that has been staged using "git add"
  • git push [alias] [branch]: pushes local commits to the remote repo into the given branch
